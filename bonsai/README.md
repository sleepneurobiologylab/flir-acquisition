# Bonsai Workflows

Bonsai workflows for dual-camera acquisition using FLIR Chameleon3 cameras via the Spinnaker SDK.

## Setup

1. Install [Bonsai](https://bonsai-rx.org/) (≥ 2.8)
2. Inside Bonsai, install the **Bonsai.Spinnaker** package via the Package Manager
3. Install the FLIR [Spinnaker SDK](https://www.flir.com/products/spinnaker-sdk/) (the C++ runtime is required even for Bonsai)

## Workflows

| File | Description |
|------|-------------|
| `dual_camera_triggered.bonsai` | Main acquisition workflow — both cameras in hardware-trigger mode |

## Camera Configuration

Each `SpinnakerCapture` node must be configured with:

- **SerialNumber** — find yours by running SpinView from the Spinnaker SDK
- **TriggerMode** — `On`
- **TriggerSource** — `Line0` (or whichever GPIO line your trigger cable is wired to)
- **TriggerActivation** — `RisingEdge`

## Notes

- The `Packages/` folder is excluded from git (generated by Bonsai's NuGet restore)
- Workflow layout files (`*.bonsai.layout`) are also excluded — they store per-machine window positions
